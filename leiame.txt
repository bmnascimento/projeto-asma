APLICATIVO
1. Pré-requisitos: é necessário ter o node com npm instalados

Link (baixar versão LTS): https://nodejs.org/en/download/


2. Dentro da pasta mobile, instalar as dependências executando o comando:

npm install


3. Instalar globalmente o expo com o comando:

npm install expo-cli --global


4. Também dentro da pasta mobile, rodar o expo com o seguinte comando: (vai abrir uma interface web com um qr code)

expo start


5. Baixar o aplicativo do expo no celular pela app store e usar o qr code para acessar o aplicativo
